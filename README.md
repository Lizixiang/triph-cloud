# triph-cloud
springcloud微服务脚手架

eureka-client------------------------注册中心客户端案例系统
eureka-server------------------------注册中心服务端系统（负责服务的注册和订阅）
triph-auth---------------------------权限管理系统（负责网关鉴权，权限拦截等）
triph-common-------------------------系统公共模块
triph-gateway------------------------网关（负责用户认证，鉴权，日志监控，限流，缓存等）
